# Louisiana Sea Level Rise Exposure

## Project Summary

This project evaluates potential sea-level rise exposure across coastal Louisiana using screening-level inundation mapping.

Three scenarios are modeled:

* 0.5 m
* 1.0 m
* 2.0 m

## Objectives

* Identify potentially exposed land
* Compare sea-level rise scenarios
* Quantify parish-level impacts
* Produce planning-oriented maps

## Data Sources

* NASADEM Elevation
* JRC Global Surface Water
* TIGER Parish Boundaries

## Methods

1. Create elevation threshold masks
2. Remove permanent water
3. Apply coastal connectivity constraints
4. Apply slope filtering
5. Generate scenario maps
6. Calculate parish exposure statistics

## Outputs

### Raster Layers

* SLR 0.5m
* SLR 1.0m
* SLR 2.0m
* Elevation Context
* Distance-to-Water Mask

### Tables

* Scenario Exposure Summaries
