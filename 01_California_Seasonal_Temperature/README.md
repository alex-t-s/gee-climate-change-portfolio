# California Seasonal Temperature Dashboard

## Project Summary

This project analyzes seasonal air temperature patterns across California using ERA5-Land climate reanalysis data.

Seasonal averages were calculated for winter, spring, summer, and fall. Temperature anomalies compare 2024 conditions against a 1991–2020 baseline.

## Objectives

* Calculate statewide seasonal temperatures
* Measure temperature anomalies
* Summarize county-level patterns
* Create an interactive dashboard

## Data Sources

* ECMWF ERA5-Land Monthly Aggregates
* TIGER/2018 State Boundaries
* TIGER/2018 County Boundaries

## Methods

1. Aggregate monthly temperature data by season
2. Convert Kelvin to Celsius and Fahrenheit
3. Calculate 1991–2020 baseline temperatures
4. Compute 2024 anomalies
5. Generate county statistics
6. Build interactive dashboard

## Outputs

### Raster Layers

* Winter Mean Temperature
* Spring Mean Temperature
* Summer Mean Temperature
* Fall Mean Temperature
* Seasonal Temperature Anomalies

### Tables

* County seasonal statistics
* Statewide summaries
