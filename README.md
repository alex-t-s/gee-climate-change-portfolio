# Climate Change GIS Portfolio

## Overview

This repository contains six geospatial analysis projects developed in Google Earth Engine (GEE) focusing on climate change, environmental monitoring, natural hazards, and landscape suitability assessment.

The portfolio demonstrates practical applications of:

* Remote Sensing
* Spatial Analysis
* Environmental GIS
* Climate Data Processing
* Raster Modeling
* Geospatial Visualization
* Google Earth Engine Development

Each project includes:

* Source code
* Exported raster datasets
* Summary tables
* Dashboard screenshots
* Methodology documentation

---

## Projects

### 01. California Seasonal Temperature Dashboard

Seasonal temperature and temperature anomaly analysis across California using ERA5-Land climate data.

**Skills Demonstrated**

* Climate data processing
* Temporal aggregation
* Anomaly analysis
* Dashboard design
* Statistical summaries

---

### 02. California Terrain-Based Solar Suitability

Terrain suitability screening for potential solar energy development using slope, aspect, and elevation.

**Skills Demonstrated**

* Multi-criteria raster modeling
* Suitability analysis
* Terrain analysis
* Renewable energy GIS

---

### 03. Everglades Hydroperiod Analysis

Assessment of long-term surface water persistence and hydrologic change across the Florida Everglades.

**Skills Demonstrated**

* Water resource analysis
* Change detection
* Environmental monitoring
* Time-series remote sensing

---

### 04. Louisiana Population Exposure

Identification of population concentrations located within low-elevation coastal areas vulnerable to sea-level rise.

**Skills Demonstrated**

* Risk mapping
* Population analysis
* Spatial weighting
* Climate vulnerability assessment

---

### 05. Louisiana Sea Level Rise Exposure

Screening-level inundation analysis under multiple sea-level rise scenarios for coastal Louisiana.

**Skills Demonstrated**

* Coastal GIS
* Hazard mapping
* Raster processing
* Scenario modeling

---

### 06. Rocky Mountain National Park Soil Erosion Risk

Simplified RUSLE-style erosion susceptibility model using rainfall, terrain, and vegetation indicators.

**Skills Demonstrated**

* Environmental modeling
* Terrain analysis
* Soil erosion assessment
* Raster suitability workflows
