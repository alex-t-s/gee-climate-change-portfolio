# California Terrain-Based Solar Suitability

## Project Summary

This project identifies terrain-based solar suitability across California using elevation, slope, and aspect derived from SRTM DEM data.

The model highlights areas with favorable terrain conditions for utility-scale solar development.

## Objectives

* Evaluate solar suitability statewide
* Develop terrain suitability index
* Identify highest-ranked locations
* Generate county summaries

## Data Sources

* NASA SRTM DEM
* TIGER County Boundaries

## Methods

1. Calculate slope
2. Calculate aspect
3. Generate elevation suitability score
4. Normalize inputs
5. Apply weighted overlay
6. Identify top 10% suitability zones

## Outputs

### Raster Layers

* Solar Suitability Index
* Slope Score
* Aspect Score
* Elevation Score
* Top 10% Suitability Zones

### Tables

* County suitability rankings
* Statewide summary statistics
