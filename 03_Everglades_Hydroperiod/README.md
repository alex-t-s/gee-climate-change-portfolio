# Everglades Hydroperiod Analysis

## Project Summary

This project evaluates long-term surface water persistence across the Florida Everglades using the JRC Global Surface Water dataset.

Hydroperiod is defined as the percentage of years classified as seasonal or permanent water.

## Objectives

* Quantify hydroperiod
* Measure historical change
* Identify significant trends
* Map hydrologic persistence

## Data Sources

* JRC Global Surface Water Yearly History

## Methods

1. Calculate hydroperiod (1984–2000)
2. Calculate hydroperiod (2001–2020)
3. Compute change between periods
4. Generate trend slope
5. Identify significant changes

## Outputs

### Raster Layers

* Hydroperiod Change
* Hydroperiod 1984–2000
* Hydroperiod 2001–2020
* Trend Slope
* Significant Trends

### Tables

* Summary statistics
