# Louisiana Population Exposure

## Project Summary

This project maps low-elevation population exposure across Louisiana to identify areas potentially vulnerable to future sea-level rise and coastal flooding.

## Objectives

* Estimate population exposure
* Identify high-risk hotspots
* Rank parishes by exposure
* Support climate adaptation planning

## Data Sources

* WorldPop Population Data
* NASADEM Elevation
* JRC Global Surface Water
* TIGER Parish Boundaries

## Methods

1. Extract low-elevation areas
2. Weight exposure by population density
3. Smooth outputs
4. Identify hotspot clusters
5. Summarize parish exposure

## Outputs

### Raster Layers

* Population Exposure
* Exposure Hotspots
* Population Base
* Elevation Weight

### Tables

* Parish Exposure Summary
