# Rocky Mountain National Park Soil Erosion Risk

## Project Summary

This project develops a simplified RUSLE-style erosion susceptibility model for Rocky Mountain National Park.

The model combines rainfall, terrain, and vegetation indicators to identify areas with elevated erosion risk.

## Objectives

* Estimate erosion susceptibility
* Identify erosion hotspots
* Classify risk categories
* Support conservation planning

## Data Sources

* CHIRPS Rainfall
* SRTM DEM
* Sentinel-2 NDVI
* WDPA Protected Areas

## Methods

1. Generate rainfall proxy
2. Calculate slope factor
3. Derive vegetation cover factor
4. Normalize inputs
5. Compute erosion index
6. Classify risk levels

## Outputs

### Raster Layers

* Classified Erosion Risk
* Continuous Erosion Surface
* Extreme Hotspots
* Slope Context
* NDVI Context

### Tables

* Summary Statistics
